主要功能：

通过磁力链接添加任务：python api.py --bittorrent-link "magnet:?xt=urn:btih:..."
通过种子文件添加任务：python api.py --bittorrent-file "temp.torrent"

配置管理：

修改IP地址：--change-qbittorrent-ipaddress
修改端口：--change-qbittorrent-port
修改用户名：--change-qbittorrent-user
修改密码：--change-qbittorrent-pass
启用/禁用日志：--change-qbittorrent-log true/false
